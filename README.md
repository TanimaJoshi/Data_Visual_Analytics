# Data_Visual_Analytics

Goal: forecast jobs in the U.S. using 10 years worth of data for the top 20 jobs 
 1. Create sql db using data gathered from the U.S. bureau of labor statistics
 2. Choose model using back testing MAPE scores
 3. Select 20 jobs by Total employee count (over all of U.S)
 4. Forecast each of these jobs for each state in the US using chosen (Prophet) model
 5. Visualize on U.S. map
